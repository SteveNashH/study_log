算法常见面试题：
1. 算法的几个特征是什么。
2. 算法复杂性的定义。大O、θ、、小o分别表示的含义。

4. 分治算法的思想，经典的分治算法(全排列、二分搜索、归并排序、快速排序、线性时间选择、最接近点对问题)。
5. 动态规划算法解题框架，动态规划算法的两个要素是什么？备忘录方法是什么？

8. 经典的贪心问题(活动安排问题、背包问题、装载问题、哈夫曼编码、单源最短路径、最小生成树问题)。
